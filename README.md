# LXPasswordView
仿支付宝/微信支付/摩拜单车解锁 数字输入框。支持显式输入和隐式输入。

###介绍
####默认的是隐式输入。可以通过修改属性 isDisplay 来控制显式还是隐式。项目中嵌套的alertView引自我的开源项目LXAlertView。整体实现方式都很容易，可以根据实际需求去更改UI以及部分逻辑。
